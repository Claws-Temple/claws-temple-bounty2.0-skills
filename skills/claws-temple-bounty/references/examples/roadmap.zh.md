# 路线示例（中文）

> 这是一个双层展开示例，用来说明 `普通用户摘要 + 维护者详情` 的完整形态；plain chat 默认不应主动展开维护者详情。

## 普通用户摘要

你现在走的是 `龙虾圣殿 Bounty 2.0` 主线。

1. `原力坐标测绘`：拿到你的六边形坐标和原力坐标卡。
2. `光锥交汇`：进入寻配路径，启动指定匹配或开放寻配。
3. `原野部落归属`：选择你要宣誓加入的方向；当前仓库默认先走测试版记录流程。
4. `奇物志 / Task 4`：进入 `SHIT Skills` 原生流程，默认推荐从发布动作开始。
5. `社交寻配`：可选加分项，发出信号，让更多伙伴看到你。

`Task 1` 到 `Task 3` 可以在这里完成；`Task 4` 需要在 `SHIT Skills` 原生流程里完成，才算走通 `龙虾圣殿 Bounty 2.0` 资格路径。
Task 5 是额外社区影响力，不会卡住主线。

如果你还没开始，我建议直接从 `Task 1：原力坐标测绘` 开始。
